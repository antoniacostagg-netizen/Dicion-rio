<div> HTML

A tag <div> é um container genérico para conteúdo de fluxo. Não tem significado específico e é usado para agrupar elementos para fins de estilização.

<div class="container">
  <p>Conteúdo aqui</p>
</div>

Uso comum: Para criar layouts e agrupar elementos.

<header> HTML

Representa um container para conteúdo introdutório ou um conjunto de links de navegação.

<header>
  <h1>Título do Site</h1>
  <nav>
    <!-- links de navegação -->
  </nav>
</header>

Uso comum: No topo da página ou como cabeçalho de seções.

<input> HTML

Usado para criar campos interativos para formulários baseados na web.

<input type="text" placeholder="Digite seu nome">
<input type="email" placeholder="seu@email.com">
<input type="password" placeholder="Sua senha">

Uso comum: Coletar dados do usuário em formulários.

Tipos comuns:

  • text - Campo de texto
  • email - Campo de e-mail
  • password - Campo de senha
  • checkbox - Caixa de seleção
  • radio - Botão de opção
display CSS

A propriedade display especifica o tipo de caixa de renderização usada por um elemento.

.elemento {
  display: block;
}

.outro-elemento {
  display: flex;
}

Uso comum: Controlar o layout e posicionamento de elementos.

Valores comuns:

  • block - Ocupa toda a largura disponível
  • inline - Permite outros elementos ao lado
  • flex - Layout flexível
  • grid - Layout em grade
  • none - Oculta o elemento
font-family CSS

Especifica a fonte a ser usada para um elemento.

body {
  font-family: 'Arial', sans-serif;
}

.titulo {
  font-family: 'Georgia', serif;
}

Uso comum: Definir a tipografia do site.

Valores comuns:

  • Nomes de fontes específicas (Arial, Times New Roman, etc.)
  • Famílias genéricas (serif, sans-serif, monospace)
  • Fontes personalizadas com @font-face
flexbox CSS

O Flexible Box Layout Module (Flexbox) torna mais fácil projetar layouts flexíveis e eficientes.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Uso comum: Criar layouts responsivos e alinhar elementos.

Propriedades principais:

  • justify-content - Alinhamento horizontal
  • align-items - Alinhamento vertical
  • flex-direction - Direção dos elementos
  • flex-wrap - Quebra de linha
